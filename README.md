# Eurican exp Banking System

A desktop banking system built with Python and Tkinter. The project demonstrates object-oriented programming concepts through a simple banking workflow with savings accounts, current accounts, transfers, transaction history, and monthly account updates.

## Features

- Desktop interface using Tkinter
- Deposit and withdrawal operations
- Transfer money between accounts
- Savings account interest update
- Current account monthly fee update
- Account summary with balance and transaction count
- Bank summary with total customers, accounts, and money
- Transaction history with timestamp, amount, and balance

## Tech Stack

- Python
- Tkinter
- Object-Oriented Programming

## Concepts Used

- Classes and objects
- Inheritance
- Abstraction
- Encapsulation
- Method overriding
- GUI event handling
- Basic input validation

## How to Run

Make sure Python is installed on your computer.

```bash
python banking_system.py
```
